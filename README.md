# MyLeafy

`MyLeafy` 是通用型校园应用，内部代码名、target 和类型命名继续使用 `Leafy`。当前首个上线校园是北京林业大学，教务连接器仍直连北林强智教务系统；App 以课表为核心，整合成绩、考试安排、教学培养、自习室、校历、社区和评教等校园场景。
