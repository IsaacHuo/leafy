# 功能地图

MyLeafy 的功能可用性由校园能力、用户身份和后端配置共同决定。以下内容描述代码库中的主要能力，不代表所有入口会在每个校园环境中显示。

## 身份与校园

- 北京林业大学强智教务登录、验证码处理与会话恢复。
- 学校会话与 Supabase 业务会话分离管理。
- 匿名业务身份、校园 profile 和教务标识的受控绑定。
- 校园能力描述、自定义校园导入和按校园裁剪入口。

## 课表与时间

- 当前周课表、周次切换、课程详情和重叠课程布局。
- 课程备注、提醒、课表背景与分享图。
- 考试、自定义倒计时和年度时间视图。
- Widget 快照、深链跳转和时间线刷新。
- 共享课表快照、一次性邀请码、只读授权和关系撤销。
- 本地缓存最近成功数据，远程刷新失败时尽量保留可读状态。

## 校园工具

| 领域 | 主要能力 |
|---|---|
| 教学培养 | 成绩、成绩分析、考试、荣誉、综合测评、教学计划与培养方案 |
| 时间日程 | 日程报告、校历、考试与自定义倒计时 |
| 学习空间 | 学习项目、资料、任务和记录 |
| 空闲教室 | 空教室查询、教室占用、热力相关能力与学习时间记录 |
| 体育 | 阳光长跑、体测记录和体育场馆 |
| 发展 | 职业规划、考研公开信息及校园扩展工具 |
| 评价 | 教师、课程等按校园配置开放的结构化评分 |

## Leafy AI

- 原生对话界面和本机对话记录。
- 用户可控制的本机学业上下文。
- 自备模型 API Key，密钥保存在 Keychain。
- 可选公开检索、网页读取和 PDF 文本转发工具链。
- 生成报告、清单、表格、流程说明等 Artifact。
- 支持导航、提醒准备和建议生成等收敛动作。

AI 不应自动修改学校原始数据、绕过确认发布社区内容，或把辅助回答表述为学校官方结论。

## 社区与通知

- 帖子流、文本与图片发布、详情、评论、点赞、投票和收藏。
- 全局置顶、分类置顶、软删除与内容状态控制。
- 站内通知、公告、反馈和社区访问限制。
- 个人资料及个人帖子、评论、点赞与投票记录。
- 分享链接和 App 深链。

## 个性化与系统集成

- 浅色/深色外观、主题色、显示密度和多语言。
- 课表背景、系统分享、Widget 与 Keychain。
- iOS 26 视觉能力按可用性增强，同时保留 iOS 17 基线体验。

## 运营能力

- React-admin 资源管理与数据总览。
- `viewer`、`operator`、`super_admin` 三类角色。
- 内容、用户、公告、反馈、目录、运行配置和管理员会话管理。
- 服务端筛选 CSV 导出、请求追踪与审计日志。
- Cloudflare Pages Functions 代理和 HttpOnly 管理会话。
